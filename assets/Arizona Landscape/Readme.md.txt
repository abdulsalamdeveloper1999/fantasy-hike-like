I have provided:
[Three styles of the same asset]
[ppi(pixel per inch) = quality or resoulution]

*Ai  file [style 1,2,3][orginal]   == orginal illustrator file (editable)
*Svg file [style 1,2,3]   == high quality web images
*Eps file [style 1,2,3]   == Encapsulated PostScript images
*Png file 300 ppi    == this image has 300 dpi quality[style 1,2,3]
*Png file 1000 ppi    == this image has 300 dpi quality[style 1,2,3]
